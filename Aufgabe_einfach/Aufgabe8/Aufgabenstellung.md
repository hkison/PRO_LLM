### **Wandle einen String in Großbuchstaben um**  

**Aufgabe:**  
Schreibe ein Programm, das einen gegebenen String in Großbuchstaben umwandelt.  

**Input Format:**  
Eine Zeile mit dem String.  

**Output Format:**  
Eine Zeile mit dem String in Großbuchstaben.  

**Beispiel:**  

**Input:**  
hello world  

**Output:**  
HELLO WORLD
